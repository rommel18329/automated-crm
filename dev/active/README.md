# Active Feature Tracking

One file per feature/fix in progress. Name files as `YYYY-MM-DD-short-description.md`.

## Template

```
# Feature: <name>
**Started:** YYYY-MM-DD
**Status:** in-progress | blocked | review

## Goal
What this feature does and why.

## Plan
- [ ] Step 1
- [ ] Step 2

## Notes
Any decisions, blockers, or context worth remembering.
```
